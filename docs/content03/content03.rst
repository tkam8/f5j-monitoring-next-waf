BIG-IP Next Central Manager API
================================================


F5 BIG-IP Next Central Manager API は、BIG-IP Next のすべてのサービスとリソースに対する F5 の単一フロントエンド API です。 F5 は、API が機能し、意味があり、業界のベスト プラクティスに従っていることを保証するように設計することに取り組んでいます。このドキュメントで説明されているオブジェクト モデルは、リソースの表示方法、構成の管理方法、および BIG-IP Next の実際のシステム状態の表示方法に関する一貫した論理的なビューを反映していると考えることができます。

本章は、API による WAF の monitoring/alerts を紹介します。
